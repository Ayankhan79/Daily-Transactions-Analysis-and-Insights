Performed end-to-end data cleaning and exploratory data analysis on a personal finance dataset using **Python** and **Pandas**. Cleaned and standardized the data, handled missing values, corrected date inconsistencies, engineered time-based features, and analyzed spending patterns through KPIs, category-wise trends, subscription costs, payment modes, and daily/monthly expense summaries. The cleaned dataset and analytical outputs are prepared for reporting and interactive **Power BI** dashboard development.
